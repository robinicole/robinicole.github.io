# The Bible(s) of statistics 
- **Pattern recognition and Machine learning** by *Steven Bishop*, this books takes a *Bayesian approach* to machine learning. The maths are explained in a very clear and detailed manner. The only disadvantage of this book is that you need to be ready to calculate high dimensional Gaussian integrals at some point.  
- **Elements of statistical learning**, you can find the free pdf on internet [here](https://web.stanford.edu/~hastie/Papers/ESLII.pdf). This books takes the *frequentist approach* to statistics and is a go-to if you want to understand all the subtleties behind regression. 

# Deep learning

* **Hands on machine learning with Scikit-learn and Tensorflow** is the first book I would advise to anybody willing to start to learn Machine learning. It is divided in two part, one on *classical machine* learning (regression, random forest, SVM,…) and another one on *deep learning* using tensorflow.
