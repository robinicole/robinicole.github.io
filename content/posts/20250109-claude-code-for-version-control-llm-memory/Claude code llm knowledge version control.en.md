When working with LLMs via the standard chatbot interface of ChatGPT, Anthropic (Claude) or another chatbot provider. One of the thing which is annoying is the lack of transparency in how the chatbot will remember your discussions between the sessions. In this article I explore how we can use Claude Code to have transparency on what the LLM remember (it is store in standard Markdown file) and how we can use git to version control this knowledge.
